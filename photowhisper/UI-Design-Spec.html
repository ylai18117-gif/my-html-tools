04 · Interaction Notes
交互细节

页面切换

Landing → Analyzing：点击上传/示例后，0.25s fade-out → 显示扫描动画。扫描线从上到下循环 2.5s，4 个分析步骤依次亮起（间隔 0.8s）。

Analyzing → Results：全部步骤完成后 0.5s → fade-in 结果页。构图网格 0.6s 淡入 → 场景图区域 0.2s 后浮现 → Tab 区域 0.3s 后浮现 → 清单卡片 0.4s 后浮现。

手势与触控

Tab Bar 横向滑动：overflow-x: auto + -webkit-overflow-scrolling: touch。选中 Tab 自动 scrollIntoView 居中。

Segment Control：点击切换俯视/侧视，SVG 图表 0.3s 交叉淡入。

按钮触控反馈：:active 状态 scale(0.96) + 无 -webkit-tap-highlight-color。
