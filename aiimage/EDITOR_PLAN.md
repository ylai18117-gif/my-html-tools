# AI 图片工坊 — 高级图片编辑功能集成方案

> 状态：待开发 | 最后更新：2026-07-10

---

## 一、目标

将 pangxiaobin/image-matting 的核心能力集成到 AI 图片工坊，替换现有不好用的高级编辑功能。

集成能力：AI 抠图、AI 修补、二次编辑（橡皮擦/涂抹恢复/移动缩放）、证件照、格式转换、图片压缩。

---

## 二、技术方案

### 2.1 AI 推理

Hugging Face 免费推理 API，模型 briaai/RMBG-1.4。

```text
前端 editor.html
↓ POST /api/matting
Cloudflare Pages Function
↓ POST https://api-inference.huggingface.co/models/briaai/RMBG-1.4
↓ Authorization: Bearer ${HF_TOKEN}
↓
返回透明背景 PNG
```

### 2.2 环境变量

Cloudflare Pages → 设置 → 变量和密钥：

| 名称 | 类型 | 值 |
|---|---|---|
| HF_TOKEN | 密钥 | Hugging Face API Token |

### 2.3 Cloudflare Functions

新增：

- `functions/api/matting.js` — AI 抠图代理
- `functions/api/inpainting.js` — AI 修补代理

## 三、页面结构

### 3.1 AI 图片工坊底部改造（aiimage/index.html）

两层结构：

```
┌──────────────────────────────────────┐
│ [比例 默认]            [更多 ▾]      │ ← 上层
├──────────────────────────────────────┤
│ [📎] [输入框................] [➤]  │ ← 下层
└──────────────────────────────────────┘
```

点击"更多"弹出功能面板：AI 抠图、AI 修补、证件照、格式转换、图片压缩。

### 3.2 高级图片编辑页（aiimage/editor.html）

独立页面，功能：

| 功能区 | 能力 |
|---|---|
| 工具栏 | 橡皮擦、涂抹恢复、移动、缩放、撤销、回退 |
| AI 抠图 | 上传图片 → 调 API → 返回透明 PNG |
| AI 修补 | 抠图后修复边缘瑕疵 |
| 证件照 | 抠图 + 选背景色 |
| 格式转换 | PNG / JPG / WebP |
| 图片压缩 | 减小文件体积 |
| 历史记录 | 所有编辑结果持久化 |

### 3.3 入口打通

- AI 图片工坊生成结果卡片 → "编辑"按钮
- AI 图片工坊历史详情 → "编辑"按钮
- 点击后带上图片跳转到 editor.html

### 3.4 编辑页历史记录

```
编辑历史列表
├── 抠图结果 → 缩略图 → 可重新加载编辑
├── 证件照 → 缩略图
└── ...

每个记录可：
  - 重新加载
  - 导出
  - 删除
  - 发送到 AI 图片工坊作为参考图
```

## 四、用户交互细节

### 场景 A：朋友做传单

```
1. 打开 AI 图片工坊
2. 上传产品照片
3. 点 [更多 ▾] → AI 抠图
4. 进入编辑页，自动抠图
5. 橡皮擦微调边缘
6. 导出 PNG
7. 回到 AI 图片工坊，抠图结果自动作为参考图
8. 输入"设计一个传单" → ChatGPT 生成
```

### 场景 B：从历史记录编辑

```
1. 打开 AI 图片工坊历史
2. 点某张生成结果 → 详情
3. 点"编辑"
4. 进入编辑页，加载这张图片
5. 抠图 / 换背景 / 压缩 / 转格式
6. 导出或回到工坊
```

### 场景 C：直接打开编辑页

```
1. 打开 aiimage/editor.html
2. 拖入图片或粘贴
3. 所有编辑功能可用
4. 历史记录自动保存
```

## 五、防重复错误

- 编辑历史用唯一 ID，不复用消息 ID
- 编辑页不用弹窗 overlay，Canvas 交互必须在完整页面
- 移动端适配双指缩放画布
- 抠图和修补并发请求不互相覆盖
- 响应式：Mobile < 768 / Tablet 768-1023 / Desktop ≥ 1024
